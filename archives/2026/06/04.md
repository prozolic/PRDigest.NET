#### 概要
JIT最適化として、ビットマスク比較を正規化する変換を追加しました。`(A & pow2) == pow2` を `(A & pow2) != 0` に、`(A & pow2) != pow2` を `(A & pow2) == 0` に統一します。右辺が常に0になるこの正規形により、後続の最適化（optimizeBools、分配変換など）がより効果的に機能するようになります。

```csharp
// 正規化前後の例
static bool Example(int foo)
{
    return ((foo & 4) == 4) || ((foo & 8) == 8);
}
// 正規化後: (foo & 12) != 0 相当に最適化される可能性
```

#### 変更内容
- **src/coreclr/jit/morph.cpp**: `fgOptimizeRelationalComparison`関数内に、`(A & pow2) ==/!= pow2`パターンの正規化ロジックを追加。マッチした場合、比較演算子を反転し右辺定数を0に変更、バリューナンバリング(VN)も同時に更新します。

#### パフォーマンスへの影響
